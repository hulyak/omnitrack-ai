## Tips

### Be Specific
✅ `"Add a supplier in Shanghai"`  
❌ `"Add a node"`

### Use Natural Language
All of these work:
- `"Add a supplier in Shanghai"`
- `"Create a supplier in Shanghai"`
- `"I need a supplier in Shanghai"`

### Combine Actions
`"Add a supplier in Tokyo, connect it to my manufacturer, and run a simulation"`

### Use Context
```
You: "Add a supplier in Shanghai"
You: "Connect it to my manufacturer"
     ↑ "it" refers to the supplier
```

### Ask for Help
- `"How do I add a warehouse?"`
- `"What's the difference between a distributor and retailer?"`
- `"Help"`

```

```

```
2. "Identify risks"
3. "Find bottlenecks"
4. "Generate report"
```

### Running Scenarios
```
1. "What if port closes?"
2. "Show mitigation options"
3. "Execute option 1"
4. "Compare results"
```
